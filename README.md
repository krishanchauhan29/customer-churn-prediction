# 🔮 Customer Churn Prediction

An end-to-end ML application to predict customer churn using **Random Forest** classifier trained on the IBM Telco Customer Churn dataset. Features an interactive dashboard with business insights and a real-time churn predictor.

## 🔴 Live Demo
👉 [Click here to view the live app](https://customer-churn-prediction292003.streamlit.app/)

## 📊 Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | 86.09% |
| AUC-ROC | 0.929 |
| Precision | 0.87 |
| Recall | 0.86 |
| F1-Score | 0.86 |

## 📌 Key Business Insights
- Month-to-Month contracts have **3x higher churn risk**
- New customers **(0-1 yr tenure)** churn the most
- Discounts and contract upgrades are the most effective retention strategies
- **Monthly Revenue at Risk** calculated per prediction

## 🛠️ Tech Stack
- **Python** — Pandas, NumPy, Scikit-learn, Imbalanced-learn
- **ML** — Random Forest, SMOTE balancing, StandardScaler
- **Streamlit** — Interactive dashboard + real-time predictor
- **Plotly** — Interactive visualizations
